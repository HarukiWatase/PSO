# watase_34_cuda.py 実行環境セットアップ

## セットアップ完了

仮想環境 `venv_cuda` が作成され、必要なライブラリがインストールされました。

## 仮想環境の使用方法

### 仮想環境のアクティベート
```powershell
.\venv_cuda\Scripts\Activate.ps1
```

### 仮想環境の非アクティベート
```powershell
deactivate
```

## インストール済みライブラリ

- **networkx** (3.6.1): グラフ操作ライブラリ
- **numpy** (2.4.1): 数値計算ライブラリ
- **cupy-cuda12x** (13.6.0): CUDA対応のNumPy互換ライブラリ（GPU加速用）

## コードの実行

仮想環境をアクティベートした後、以下のコマンドで実行できます：

```powershell
python watase_34_cuda.py
```

## 注意事項

1. **Pythonバージョン**: 現在の仮想環境はPython 3.11で作成されています。Python 3.12が必要な場合は、手動でPython 3.12をインストールしてから、以下のコマンドで新しい仮想環境を作成してください：
   ```powershell
   py -3.12 -m venv venv_cuda_312
   ```

2. **CuPy**: CuPyはCUDA 12.x対応版をインストールしています。CUDAが利用できない環境では、コードは自動的にNumPyにフォールバックします。

3. **実行時間**: このコードは大規模なシミュレーションを実行するため、完了までに時間がかかる場合があります。

## トラブルシューティング

### CuPyが動作しない場合
CuPyがインストールされていても、CUDAが利用できない場合は、コードは自動的にNumPyを使用します。警告メッセージが表示されますが、実行は可能です。

### モジュールが見つからないエラー
`create_graph_4` や `savef` が見つからない場合は、これらのファイルが同じディレクトリにあることを確認してください。
